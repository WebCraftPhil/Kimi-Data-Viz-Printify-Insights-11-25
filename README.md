## ğŸ“– Project Overview
This project transforms raw PDF sales exports from a Printify print-on-demand business into a dynamic, responsive web application. It serves as a case study in **rapid data visualization** and **automated business intelligence**.

Instead of static spreadsheets, this dashboard provides interactive insights into:
* **Revenue Trends:** Weekly tracking with 4-week moving averages to smooth volatility.
* **Seasonality:** Identification of key sales windows (specifically the Q2 Spring peak).
* **Product Performance:** Breakdown of top-performing SKUs (T-Shirts & Hoodies).
* **Growth Trajectory:** Cumulative revenue analysis over the 2025 fiscal period.
